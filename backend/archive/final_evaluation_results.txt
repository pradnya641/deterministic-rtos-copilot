Input impedance is a measure of how much the input circuit's resistance affects the accuracy of an Analog-to-Digital Converter (ADC).

**How input impedance affects ADC accuracy:**

* **High input impedance:**
    * The input resistance is high, meaning the input current has significant drop across it.
    * This can lead to measurement inaccuracies due to noise and reduced signal-to-noise ratio.
    * The input current may also be affected by the ADC's internal circuitry, reducing its effective input impedance.
* **Low input impedance:**
    * The input resistance is low, and the input current is drawn directly from the source driving the ADC.
    * This can improve the accuracy of the measurement, as the input current is essentially unaffected by the ADC's internal circuitry.
    * However, low input impedance can also cause distortion and reduced resolution of the ADC readings.

**Factors affecting input impedance:**

* **Source impedance:** The impedance of the source driving the input signal also affects the input impedance.
* **Input buffer type and settings:** The internal buffer used to buffer the input signal can also influence its input impedance.
* **PCB layout and component placement:** The PCB layout and component placement can also affect the input impedance.

**Example:**

* If the input signal source has a high impedance (e.g., 10 kΩ), the input impedance of the ADC will be relatively high, leading to measurement inaccuracies.
* However, if the input buffer has a low impedance (e.g., 100 Ω), the input impedance will be lower, resulting in improved accuracy.

**Conclusion:**

Input impedance is a critical parameter that affects ADC accuracy. A high input impedance leads to measurement inaccuracies, while a low input impedance can introduce distortion and reduced resolution. Understanding the factors affecting input impedance is essential for achieving accurate ADC readings.

QUESTION 11: Why must ADC clock be <= 4.5 MHz?
--------------------------------------------------
STATUS: success
TIME: 2.05s

ANSWER:
### 🧠 Hardware Insight: ADC Clock Constraint
The LPC2148 ADC hardware requires the clock frequency to be **≤ 4.5 MHz**.

**Reasoning Chain:**
- The Successive Approximation (SAR) logic requires a specific settling time per bit.
- If the clock is too fast, the internal comparator cannot settle, leading to non-linearities and bit errors.

**Hardware Result:**
- **Symptom:** Unstable or completely random digital output values.
- **Fix:** Set CLKDIV in AD0CR such that `PCLK / (CLKDIV + 1) <= 4.5 MHz`.

**ADC (Analog-to-Digital Converter):**

The ADC is an integrated circuit or microcontroller feature used to convert continuous analog signals into discrete digital values. This allows digital systems to read and process analog measurements such as voltage, current, pressure, and temperature.

The LPC2148 has multiple ADCs, each with its dedicated conversion channel and digital output. These ADCs can be configured to operate in different modes, including:

* **Single-ended mode:** This mode is suitable for single-ended analog signals with a maximum voltage of 5V.
* **Differential mode:** This mode is used for differential signals where the two inputs are connected together.
* **Bridge mode:** This mode is used for connecting multiple inputs together, allowing for multiple readings from a single analog source.

**Steps to Configure an ADC:**

1. **Select the ADC pin:** Choose the appropriate ADC pin based on the application. The available pins are typically marked on the PCB or in the datasheet.
2. **Set the configuration bits:** Use the appropriate registers or settings to configure the ADC mode, resolution, reference voltage, and other parameters.
3. **Start the ADC conversion:** Once the configuration is complete, start the ADC conversion process.
4. **Read the digital value:** Once the conversion is complete, read the digital value from the corresponding register.
5. **Process and use the digital value:** The digital value can be used directly or further processed depending on the application.

**Additional Information:**

* The resolution of an ADC is the number of bits used to represent the input value. The higher the resolution, the more accurate the conversion.
* The reference voltage is the voltage used to amplify the analog signal before conversion.
* The sampling rate determines the frequency at which the ADC takes readings.

**Example:**

To configure the ADC on pin A0 for single-ended mode with a resolution of 8 bits and a reference voltage of 5V:

```c
// Configure ADC on pin A0

ADCSREG |= 0x04; // Set resolution to 8 bits
ADCSREG |= 0x02; // Set reference voltage to 5V
```
